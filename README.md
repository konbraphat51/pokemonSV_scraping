# pokemonSV_scraping
Getting the data below of Pokemon SV from GameWith.
* Base Status
* Link for Pokemon's image
* type(Japanese)
* Move to remember (Japanese)
* Characteristic (Japanese)

And also get Jp -> Eng translation data of:
* Pokemon Name
* Moves
* Characteristics

GameWithからポケモンSVの各種データを取得するコードです。
* 種族値
* ポケモンの画像URL
* タイプ
* 覚える技
* 特性
